# Eat Da Burger!
Eat Burgers using 

<ul>
<li>Node.js</li>
<li>Express</li>
<li>MySQL</li>
<li>Handlebars</li>
</ul>



## Usage

Add a burger name to the menu / database. Eat burgers by clicking on it in the left column. Burgers in the right column have already been eaten.
